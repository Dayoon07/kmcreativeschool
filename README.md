---

### 경찰등록
- **파일 업로드 (프로필사진)**
- **경찰서 등록 (공공 API)**

---

### 신고
- **게시글 신고 (SMTP로 전송)**
- **채팅으로 신고 (파일 업로드 처리)**
- **신고자 등록**
- **가해자 등록**

---

### 신고처리
- **신고내용 조회**
- **신고내용 접수 및 완료**
- **SMTP 상황 전송**
